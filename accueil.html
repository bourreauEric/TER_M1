Animation d'automates

Bonjour, cette application est conçu pour vous afin de vous aider à dessiner proprement votre automate à partir d'un document text respectant une structure bien definie, de même vous pouvez tester la reconaissance de mots et ceci avec une jolie animation.

Télécharger le code source

Démonstration

Vous pouvez lancer une démonstration en chargeant un fichier text respectant la syntaxe qui se trouve dans l'onglet Syntaxe

Après avoir chargé le fichier et si la syntaxe est juste vous allez voir apparaitre le dessin d'un automate que vous pourrez manipuler à la suite.

Syntaxe

Afin que votre votre fichier soit compilable, il faut respecter la syntaxe requise et ceci en indiquant les elements suivants :

  • le nombre d'états
  • la liste des etiquettes
  • l'état d'entrée
  • l'état de sortie
  • la liste des transitions
